Struktur Proyek:

├── index.html         # Halaman utama
├── about.html         # Halaman tentang
├── projects.html      # Halaman proyek
├── contact.html       # Halaman kontak
├── styles/
│   └── style.css      # File CSS utama
├── scripts/
│   └── form-handler.php # Script PHP untuk mengolah formulir kontak
└── README.txt         # File ini

Cara Menjalankan:

Kloning Repositori: Kloning proyek ini ke komputer kita.
Server Lokal: Pastikan sudah memiliki server lokal seperti XAMPP atau WAMP karena file PHP memerlukan server untuk diproses.
Buka di Browser: Buka index.html di browser untuk melihat website.
